# Apex Wrench Solutions - Professional Plumber Website

## Overview
A complete website for Apex Wrench Solutions - a professional plumbing service company.

## Features
1. **Portfolio/Gallery** - Display videos and pictures of previous work
2. **Booking System** - Online appointment booking form
3. **Services** - Complete list of plumbing services offered
4. **Reviews** - Customer reviews and ratings
5. **Admin Portal** - Manage portfolio, bookings, reviews, and settings

## Files
- `index.html` - Main website
- `login.html` - Admin login page
- `admin.html` - Admin dashboard
- `script.js` - Website JavaScript
- `admin.css` - Admin styles
- `styles.css` - Website styles
- `my logo.png` - Company logo

## How to Use
1. Open index.html in a browser to view the website
2. Open login.html to access admin portal
3. Login with admin/apex2024
4. In admin, use Settings to upload a new logo or change password
5. Use Upload Media to add portfolio items
6. View bookings and reviews in their respective sections
